# Flask

Install on Windows::
```sh
> pip install flask
```

Install on Pi:
```sh
> sudo apt-get install python3-flask:
```

Run on windows:
https://stackoverflow.com/questions/31080829/python-error-io-unsupportedoperation-fileno

```sh
> python3 app.py
```
